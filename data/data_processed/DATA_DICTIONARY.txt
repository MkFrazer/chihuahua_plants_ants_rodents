This Data Dictionary explains the file formatting for the CLEANED DATA for the project, 
"Plant Species Diversity as a function of ant and rodent predation in the Chihuahua desert."

* plants_clean contains the following variables:
    Year: year the plot was surveyed
    Plot: plot number (1-24)
    total_species: how many species were observed or had an observation count of 0
    total_abundnace: the total number of plants on that plot
    avg_abundance: the average number of plants per species per plot per year
  
* precip_clean contains the following variables:
    Year: year the precipitation measurements were taken
    Annual_avg_precip: the average amount of precipitation, calculated from monthly data
    
    Note: Precipitation for all plots came from one precipitation sensor
    
* treatments contains the following variables:
    PLOT: plot number (1-24)
    YEAR: year the plot was treated
    LARGE_RODENTS: binary variable representing the exclusion or inclusion of kangaroo rats. 
        0 = excluded (not present)
        1 = not excluded (present)
    ALL_RODENTS: binary variable representing the exclusion or inclusion of all rodents. 
        0 = excluded (not present)
        1 = not excluded (present)
    PORU_ANTS: binary variable representing the exclusion or inclusion of Pogonomyrmex rugosus,
      a dominant granivorous ant species. 
        0 = excluded (not present)
        1 = not excluded (present)
    ALL_ANTS: binary variable representing the exclusion or inclusion of ant colonies of all species. 
        0 = excluded (not present)
        1 = not excluded (present)
